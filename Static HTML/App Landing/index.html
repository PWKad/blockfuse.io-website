Mission Statement

Building developer-focused solutions for Litecoin and Bitcoin App development.

Values

  • Simplicity

    Reducing the technical barrier to building on top of Litecoin and Bitcoin is of primary importance.
  • Blockchain Agnostic

    Acknowledge that adaptive tools are necessary as the space develops towards a multicoin future.
  • Open Source

    In the spirit of decentralization, make source code available to be verified through peer review and unrestricted to match the exponential growth of this space.
  • Interoperability

    Ensure applications are compatible with one another across the various reference clients available for Litecoin and Bitcoin.
Developer Focused

Prioritize the needs of developers to create consumer facing applications easier.

Why Choose Us

Let's be honest. Building applications on top of Litecoin and Bitcoin is difficult. Even if you are a top-notch programmer, in-depth protocol level knowledge is required when using or adapting a blockchain client for your product. The enourmous level of time and knowledge needed to learn these intricacies can be overwhelming barrier for new developers. Here at BlockFuse, we see a great need in this space to make this entry point more accessible. In light of this, we will remove the need for protocol level development knowledge needed by offering tools across several reference clients such as litecoind, bitcoind, ltcd, and even btcd. We will also open source these tools so as not to restrict or limit the growth of these ecosystems. In this way, we hope to facilitate application development in a way that can match the exponential growth of these two currencies.
